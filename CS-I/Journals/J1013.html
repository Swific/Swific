2.1: Why is the formula for determining the number of rows in a truth table related only to the number of inputs, and why is the formula 2^n?
The formula for determining the number of rows in a truth table is related only to the number of inputs since the number of rows depends on the amount of inputs there are. The formula is 2^n since each variable can either be true or false (0 or 1) and there are also "n" inputs for the truth table.
